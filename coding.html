Java

標準入出力

最初に、 

Scanner in = new Scanner(System.in);
（必ずしも in という変数名である必要はないのでお好みで）
int 型の変数 n に入力された整数値を入れたい時は、 
n = in.nextInt();
String 型の変数 str に入力された単語を入れたい時は、 
str = in.next();
1行(次の改行まで)の場合は、 
str = in.nextLine();

出力は、 
System.out.print("Hello,");
System.out.println(" World!");
print では改行が行われず、次の出力が同じ行に出力される。
問題文をよく読み、改行が必要か判断すること。

配列・ループ

要素数が10個の int 型の配列 arr を作るには、 

int[] arr = new int[10];
実際に使えるのは arr[0] から arr[9] なので注意(1から10ではなく0から9)。

n 回処理を繰り返すには、 
for (int i = 0; i < n; ++i) {
	// 処理
}
例えば、arr の中身をカンマ区切りですべて出力するコードは、 
for (int i = 0; i < arr.length; ++i) {
	if (i != 0) {
		System.out.print(",");
	}
	System.out.print(arr[i]);
}
「カンマ区切りで出力する」＝「2つ目以降の要素の前にカンマを付ける」

その他

文字列関係 
String str = "abcdef";
int len = str.length(); // len = 6;
char c = str.charAt(1); // c = 'b';
String sub = str.substring(2, 5); // sub = "cde";
String a = "cde";
if (a.equals(sub)) { // 文字列が等しいか(a == sub ではないので要注意！)
	System.out.println("a と sub は同じ！");
}

型
  • int: 整数
  • double: 小数(浮動小数点数)
  • String: 文字列
  • char: 文字

C

標準入出力

int n;
scanf("%d", &n); // 整数を1つ入力
printf("%d\n", n); // n の値を出力して改行

C++

標準入出力

int n;
cin >> n; // 整数を1つ入力
cout << n << endl; // n の値を出力して改行
