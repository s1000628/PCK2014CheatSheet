エディタ・IDE

Windows

Java(Emacs)

Emacs を起動する。
プロジェクトを作成する。
クラスを作成する(例：prob01.java)。この時、main メソッドを自動生成する。
コードを書く。
実行する。
入力をあらかじめメモ帳などに貼り付けておくと、コピペで入力できるので時間の短縮になります。

C, C++(Visual Studio)

Visual Studio を起動する。
プロジェクトを作成する。
main.cpp を作成する(C の場合も .cpp でOK)。
コードを書く。
入力をあらかじめメモ帳などに貼り付けておくと、コピペで入力できるので時間の短縮になります。

UNIX

Java(javac)

好きなエディタでコードを書く(例:prob01.java)。
コマンド 

javac prob01.java
でコンパイルする。エラーが無ければ prob01.class が生成される。
コマンド 
java prob01
で実行。
入力をあらかじめテキストファイルにしておき、リダイレクションを使うと楽です。
java prob01 < input01a.txt

C(gcc)

好きなエディタでコードを書く(例:prob02.c)。
コマンド 

gcc prob02.c -o prob02
でコンパイルする。エラーが無ければ prob02 が生成される。
コマンド 
./prob02
で実行。
入力をあらかじめテキストファイルにしておき、リダイレクションを使うと楽です。
./prob02 < input02a.txt

C++(g++)

好きなエディタでコードを書く(例:prob03.cpp)。
コマンド 

g++ prob03.cpp -o prob03
でコンパイルする。エラーが無ければ prob03 が生成される。
コマンド 
./prob03
で実行。
入力をあらかじめテキストファイルにしておき、リダイレクションを使うと楽です。
./prob03 < input03a.txt
