エディタ・IDE

Windows

Java(Eclipse)

  1. Eclipse を起動する
    • スタートメニュー → すべてのプログラム → 授業用ソフト → eclipse3.7
  2. ワークスペースを設定する
    1. ファイル → ワークスペースの切り替え → その他
    2. 参照ボタンを押し、「ホームフォルダ(Z:)」に「PCK2014」というフォルダを作り、それを選択
    3. OKボタンを押すと Eclipse が再起動する
      • ようこそ画面が出たらそのタブを閉じる。
  3. プロジェクトを作成する
    1. ファイル → 新規 → Java プロジェクト
    2. プロジェクト名(例: PCK2014_01)を入力して完了ボタンを押す
    • これで「パッケージ・エクスプローラー」にプロジェクトが表示される。
  4. クラスを作成する
    1. パッケージ・エクスプローラーのプロジェクトを右クリック → 新規 → クラス
    2. 名前(例: Prob01)を入力
    3. 「public static void main(String[] args)」にチェック
    4. 完了ボタンをクリック
  5. コードを書く
  6. 実行する
    • Ctrl + F11 キーで実行できる。
    • 「コンソール」画面が見当たらない場合は、メニューからウィンドウ → ビューの表示 → コンソール。
    • 入力をあらかじめメモ帳などに貼り付けておくと、コピペで入力できるので時間の短縮になる。

C, C++(Visual Studio)

  1. Visual Studio を起動する
    • スタートメニュー → すべてのプログラム → Microsoft VisualStudio 2010 Express → Microsoft Visual C++ 2010 Express
  2. プロジェクトを作成する
    • 最初に作る場合
      1. ファイル → 新規作成 → プロジェクト
      2. 「Win32 コンソール アプリケーション」を選択
      3. 「名前」にプロジェクト名(例: prob01)を入力
      4. 「場所」の参照ボタンを押して「Z:\PCK2014\」を選択
      5. 「ソリューション名」に「PCK2014」と入力
      6. OKボタンをクリック
      7. Win32 アプリケーション ウィザードが表示されたら次へをクリック
      8. 「空のプロジェクト」にチェックを入れる
      9. 完了ボタンをクリック
        • ここで「参照データベースと...見つかりました」というウィンドウが出たらOKボタンを押す
    • 2つ目以降を作る場合
      1. ファイル → 新規作成 → プロジェクト
      2. 「Win32 コンソール アプリケーション」を選択
      3. 「名前」にプロジェクト名(例: prob02)を入力
      4. 「ソリューション」を「ソリューションに追加」に変更
      5. OKボタンを押す
      6. あとは上記と同じ
  3. ソースファイルを作成する
    1. ソリューションエクスプローラーのプロジェクト(例: prob01)を右クリックし、追加 → 新しい項目
    2. 「C++ ファイル(.cpp)」を選択
      • C言語で書く場合もこれでOK。
    3. 「名前」に「main」と入力
    4. 追加ボタンを押す
  4. コードを書く
  5. 実行する
    • Ctrl + F5 キーで実行できる。
    • 入力をあらかじめメモ帳などに貼り付けておくと、コピペで入力できるので時間の短縮になる。
    • コマンドプロンプトにコピペするときは右クリックで貼り付け。Ctrl+V は使えない。
  • 注意
    • 環境によっては、実行後に結果を見る間もなく実行画面が消えてしまう場合があるようです。
      その場合は、
      • main 内の最後の行にカーソルを合わせて F9 キーを押し(ブレークポイントを設定し)、F5 キーで実行する。
      • コマンドプロンプトから直接実行する。
      のいずれかで対応してください。

Linux

Java(javac)

  1. 好きなエディタでコードを書く。 
    vim Prob01.java
  2. コンパイルする。 
    javac Prob01.java
    • エラーが無ければ Prob01.class が生成される。
  3. 実行する。 
    java Prob01
    • 入力をあらかじめテキストファイルにしておき、リダイレクションを使うと楽。 
      java Prob01 < input01a.txt

C(gcc)

  1. 好きなエディタでコードを書く。 
    emacs prob02.c
  2. コンパイルする。 
    gcc prob02.c -o prob02
    • エラーが無ければ prob02 が生成される。
  3. 実行する。 
    ./prob02
    • 入力をあらかじめテキストファイルにしておき、リダイレクションを使うと楽。 
      ./prob02 < input02a.txt

C++(g++)

  1. 好きなエディタでコードを書く。 
    gedit prob03.cpp
  2. コンパイルする。 
    g++ prob03.cpp -o prob03
    • エラーが無ければ prob03 が生成される。
  3. 実行する。 
    ./prob03
    • 入力をあらかじめテキストファイルにしておき、リダイレクションを使うと楽。 
      ./prob03 < input03a.txt
