エディタ・IDE

Windows

Java(Eclipse)

  1. Eclipse を起動する。
  2. プロジェクトを作成する。
  3. クラスを作成する。
    • 例: Prob01
    • main メソッドを自動生成するようにチェックするといい。
  4. コードを書く。
  5. 実行する。
    • 入力をあらかじめメモ帳などに貼り付けておくと、コピペで入力できるので時間の短縮になります。
    • コマンドプロンプトにコピペするときは、右クリックで貼り付けます。Ctrl+V は使えません。

C, C++(Visual Studio)

  1. Visual Studio を起動する。
  2. プロジェクトを作成する。
  3. ソースファイルを作成する。
    • 例: main.cpp
    • C の場合も .cpp で OK。
  4. コードを書く。
    • 入力をあらかじめメモ帳などに貼り付けておくと、コピペで入力できるので時間の短縮になります。
    • コマンドプロンプトにコピペするときは、右クリックで貼り付けます。Ctrl+V は使えません。

UNIX

Java(javac)

  1. 好きなエディタでコードを書く。 
    vim Prob01.java
  2. コンパイルする。 
    javac Prob01.java
    • エラーが無ければ Prob01.class が生成される。
  3. 実行する。 
    java Prob01
    • 入力をあらかじめテキストファイルにしておき、リダイレクションを使うと楽です。 
      java Prob01 < input01a.txt

C(gcc)

  1. 好きなエディタでコードを書く。 
    emacs prob02.c
  2. コンパイルする。 
    gcc prob02.c -o prob02
    • エラーが無ければ prob02 が生成される。
  3. 実行する。 
    ./prob02
    • 入力をあらかじめテキストファイルにしておき、リダイレクションを使うと楽です。 
      ./prob02 < input02a.txt

C++(g++)

  1. 好きなエディタでコードを書く。 
    gedit prob03.cpp
  2. コンパイルする。 
    g++ prob03.cpp -o prob03
    • エラーが無ければ prob03 が生成される。
  3. 実行する。 
    ./prob03
    • 入力をあらかじめテキストファイルにしておき、リダイレクションを使うと楽です。 
      ./prob03 < input03a.txt
